👉 Проблема в тому, що ти зробив:

CSS

fill: none;

Але ця іконка спочатку була “залита” (fill), а не намальована лініями.
Коли ти прибираєш fill, браузер починає показувати контур SVG, включаючи його bounding box — виглядає як квадрат.

🔧 Як це виправити

✅ Варіант 1 (рекомендований): використовуй color, а не stroke/fill

CSS
&_logo_icon {
    color: #ccff00;
}

👉 Це найправильніше для react-icons — вони зазвичай підлаштовані під color.

✅ Варіант 2: залишити fill

CSS
&_logo_icon {
    fill: #ccff00;
}

⚠️ Варіант 3: якщо хочеш саме outline-іконку

Тоді треба брати іншу іконку, наприклад з іншого набору:

  • MdOutlineVideocam

  • або з Heroicons / Feather

JavaScript
import { MdOutlineVideocam } from "react-icons/md";

💡 Чому з’являється квадрат

SVG має:

  • viewBox (прямокутна область)

  • path всередині

Коли ти ставиш fill: none, але додаєш stroke, то:

  • браузер може обвести весь path (іноді включаючи рамки)

  • і це виглядає як квадрат

✔️ Висновок

  • ❌ stroke + fill: none → не для всіх іконок

  • ✅ використовуй color

  • або бери outline-версію іконки

Якщо хочеш — можу підібрати тобі стильні іконки під твій UI (щоб виглядали як YouTube/modern SaaS 😏)

  • 🔹 Саму форму іконки скруглити не можна (без зміни SVG)
